# Random Joke Page

Welcome to the **Random Joke Page** repository! This project is a simple web page that displays random jokes using HTML, CSS, and JavaScript.

## Features

- **Random Jokes**: The page fetches jokes from an API and displays them randomly.
- **Responsive Design**: The layout adjusts to different screen sizes.

## Getting Started

1. **Clone the Repository**: To get started, clone this repository to your local machine:

    ```bash
    git clone https://github.com/scary-sherlock/Random-Joke-Page.git
    ```

2. **Open `index.html`**: Open the `index.html` file in your web browser to view the random jokes.
